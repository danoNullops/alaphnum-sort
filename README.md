# AlphaNum Sort

Simple alphanumeric sorting algorithm that sorts device names from file.
Could be used to sort any string that contains numbers this way.

## Why

To sort device names like so:

switch-t4-r15
switch-t5-r15
switch-t9-r1
switch-t10-r1
switch-t10-r5
switch-t11-r1
switch-t12-r34
switch-t22-r1

Instead of:

switch-t1-r1
switch-t10-r1
switch-t10-r5
switch-t11-r1
switch-t12-r34
switch-t2-r1
switch-t2-r22
switch-t22-r1
switch-t4-r15

## Prerequisites

Node and npm are required.

[Install Node/npm](https://nodejs.org/en/download/package-manager/)

## Install

```
npm install
```

## Run

```
npm start
```

## Tests

Run tests with:

```
npm test
```
